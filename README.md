# Holberton School - Unix: Shell, I/O Redirection and Filters

## 📝 Description
This repository contains shell scripts and projects focused on mastering the Unix command line. The primary goal is to understand how the shell handles data flow using I/O redirection, pipes, and various text manipulation filters.

## 🛠 Project Overview
The tasks within this repository cover:
* **I/O Redirection:** Handling standard input (`stdin`), standard output (`stdout`), and standard error (`stderr`).
* **Piping:** Chaining multiple commands together to perform complex data processing.
* **Special Characters:** Understanding the shell's interpretation of wildcards and expansion.
* **Data Manipulation:** Using core Unix utilities to filter, sort, and transform text data.

## 💻 Commands & Utilities Used

| Command | Function |
| :--- | :--- |
| `echo` | Displays a line of text or a variable. |
| `cat` | Concatenates and displays file content. |
| `head` / `tail` | Outputs the first or last part of a file. |
| `find` | Searches for files in a directory hierarchy. |
| `wc` | Prints newline, word, and byte counts for a file. |
| `sort` / `uniq` | Sorts lines of text and filters out repeated lines. |
| `grep` | Prints lines matching a specific pattern. |
| `tr` / `cut` | Translates/deletes characters or removes sections from lines. |
| `rev` | Reverses the characters of each line. |
| `passwd (5)` | Understanding the structure of the `/etc/passwd` file. |

## 🚀 Learning Objectives
* How to use redirection and pipes to create efficient workflows.
* Mastering the use of special characters in the shell.
* Learning the basics of system administration through files like `/etc/passwd`.
* Writing clean, executable Bash scripts.

## 📂 Requirements
* All scripts are tested on **Ubuntu 20.04 LTS**.
* All files start with the shebang: `#!/bin/bash`.
* All scripts are executable.
